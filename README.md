# 🚀 SchemFlow

**SchemFlow** is a precision schematic annotation tool designed for engineers to draw, edit, and export clean wire overlays on circuit diagrams and PCB images.

🔗 Live App: https://schemflow.netlify.app

---

## ✨ Features

* 🎯 **Precision Wire Drawing**

  * Draw accurate wire traces over schematics and PCB layouts

* 🔄 **Multiple Routing Modes**

  * Direct (free line)
  * PCB Mode (90° orthogonal routing)

* 🎨 **Rendering Styles**

  * Smooth (rounded curves, clean visuals)
  * Rigid (sharp corners, full node visibility)

* 🧠 **Smart Interaction**

  * Click to start
  * Double-click to finish wiring
  * Undo support

* 🖼️ **Image Support**

  * Upload PNG, JPG, WEBP (up to 50MB)

* 📤 **Export Ready**

  * Export annotated schematic with overlays

---

## 🛠️ Tech Stack

---

## 📦 How to Use

1. Open the app
2. Upload your schematic or PCB image
3. Select **Wire Tool**
4. Choose routing mode (Direct / PCB)
5. Draw wires:

   * Click to place points
   * Double-click to finish
6. Adjust style and thickness
7. Export final annotated image
